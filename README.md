# C-Sudoku-Project

# Правила "Судоку"
• Необходимо заполнить каждый столбик, каждую строку, и каждый блок 3 на 3 цифрами от 1 до 9 без повторений

# Управление
• Выбор поля мышью
• 0-9 (сверху или на нумпаде)

# Функциональность
• генерация поля с единнственным решением
• решение по средствам итераций
• ввод своего судоку 
• сохранение и загрузка игры

# Требования для запуска
Легче всего запусить используя архив sudoku.zip, распакуйте, запустите sudoke.exe. Работает только на Windows.
В случае использования другой ОС, придётся компилировать самому :C, требуется SFML 3.0.2.
